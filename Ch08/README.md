# just-cool-button

## What is it?

This is a cooooool button!

## How to use?

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import JustCoolButton from 'just-cool-button';
 
ReactDOM.render(
  <JustCoolButton style={{ fontSize: 32, }} />,
  document.getElementById('root')
);
```
